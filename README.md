# p0ckit.github.io

Official website for the **p0ckit** framework - a modular Bash-based security toolkit.

## About

This is the GitHub Pages site for [p0ckit](https://github.com/p0ckit/p0ckit), a lightweight offensive security framework designed for penetration testers, security researchers, and red teamers.

The site showcases:
- **Framework Overview** - What p0ckit is and why you should use it
- **Core Modules** - Documentation on built-in security modules
- **Installation Guide** - Setup instructions for Linux and Docker
- **Usage Examples** - Step-by-step guides to get you started
- **Custom Module Creation** - How to build your own modules

## Site Features

- 🎨 Modern dark theme with cyan/magenta accent colors
- 📱 Fully responsive design for mobile and desktop
- 🚀 Fast, lightweight performance
- ⚡ Interactive navigation and smooth animations

## Local Development

To view this site locally:

```bash
# Clone the repository
git clone https://github.com/p0ckit/p0ckit.github.io.git
cd p0ckit.github.io

# Open index.html in your browser
# Or use a local server:
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.
