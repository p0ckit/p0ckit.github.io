A detailed list of all currently available security modules in the p0ckit core.

Core Modules

core/maccng

No description available.

core/ntscan

this is a script that just runnes nmap -sV with the given ip and port

Options:

  • rhost (required)
  • port (optional)

core/sok_scan

Set the ip address to scan

Options:

  • --ip: Set the ip address to scan
  • --port: Specify the port or port range to scan
  • --timeout: Set the timeout value
  • --workers: Set the amount of workers

core/sub_enum

this is a subdomain enumerator made in python

Options:

  • required:
  • --url | --uw/--url-wordlist
  • -s / --subdomain | -w / --wordlist
  • optional:
  • -d / --delay
  • -wr / --worker

core/web_fuzzer

this script is a web fuzzer made in python that helps try endpoint's and url's giving the status code e.g. 200

Options:

  • required:
  • --url | --uw / --url-wordlist
  • -e / --endpoint | -w / --wordlist
  • optional:
  • -f / --filter
  • -t / --timeout
  • -d / --delay
  • for more use "set -h/--help something" (it will work but the key needs a value to register it)

core/wifi_attack

This is a aircrack-ng wrapper made in bash to ease wifi attacks

Options:

  • iface - required
  • type - required (set the type of the scan you want: recon, deauth, hand_cap)
  • bssid - required (for all types besides recon)
  • count - optional (set the number deauth pkg sent, default is until stoped)
  • client - optional (set a specific client for deauth)
  • channel - required (set the channel of AP (required cause sometimes aireplay-ng doesn't get the right channel))
