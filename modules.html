A detailed list of all currently available security modules in the p0ckit core.

Core Modules

core/ntscan

A Bash-based wrapper for nmap. Performs service version detection on a target host.

Options:

  • rhost (required): Target IP or hostname
  • port (optional): Target port

core/wifi_attack

Comprehensive wrapper for the aircrack-ng suite. Chains reconnaissance, handshake capture, and cracking.

Options:

  • interface (required): Wireless interface (e.g., wlan0mon)
  • attack_type (required): attack type

core/web_fuzzer

Python-based web endpoint fuzzer. Scans for hidden directories and files.

Options:

  • url (required): Target URL
  • threads (optional): Number of concurrent threads

core/sub_enum

Subdomain enumeration tool for identifying subdomains of a target domain.

Options:

  • domain (required): Target domain
  • threads (optional): Number of concurrent threads

core/sok_scan

Lightweight port scanner built in pure Python3. High-speed scanning without external dependencies.

Options:

  • range (required): IP range (e.g., 192.168.1.1-192.168.1.50 or 192.168.1.0/24)
